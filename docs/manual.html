Visao geral

TermiCanvas organiza terminais e agentes CLI em um canvas visual para acompanhar execucoes, dividir trabalho e coordenar agentes locais.

pip install -r requirements.txt
python main.py

Canvas

Use o canvas para posicionar nodes como em um quadro infinito.

  • Pan: botao do meio ou Espaco + arraste.
  • Zoom: Ctrl + roda.
  • Snap: grid de 40 px; segure Alt para movimento livre.
  • Atalhos: Ctrl+T, Ctrl+W, Ctrl+Tab, Ctrl+Shift+D.

Terminais

Crie PowerShell ou CMD pela Tool Island. O terminal usa PTY real, scrollback e resize responsivo.

  • Ctrl+C com selecao copia.
  • Ctrl+C sem selecao interrompe o shell.
  • Ctrl+V cola.
  • Shift + roda navega pelo scrollback.

Agentes

Claude, Codex e Gemini rodam como CLIs dentro de terminais. No dialogo, escolha PowerShell ou CMD como shell base.

  • Claude usa CLAUDE.md.
  • Codex usa AGENTS.md.
  • Gemini usa GEMINI.md.

Bus e CLI

O bus local permite que agentes troquem mensagens por comandos Python.

python -m termicanvas.cli list
python -m termicanvas.cli send <node_id> "mensagem"
python -m termicanvas.cli inbox
python -m termicanvas.cli spawn codex "Builder" --role-file role.md

Orquestracao

Promova um agente a orquestrador para ele criar agentes filhos, delegar tarefas e acompanhar respostas via inbox.

Agentes criados por spawn recebem pasta isolada em .termicanvas/<slug>/ e uma chain visual no canvas.

Widgets

  • Nota: texto livre no canvas.
  • Prompt Card: envia texto para outro node conectado.
  • Agent Widget: chamada one-shot para perguntas rapidas.

Snapshots

Snapshots salvam o estado do canvas em ~/.termicanvas/snapshots/. Use Ctrl+S para salvar.

Debug Monitor

O Debug Monitor mostra RAM, CPU, timers, erros, threads, render times e metricas por terminal. Abra com Ctrl+Shift+D.
